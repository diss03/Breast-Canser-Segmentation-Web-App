## Описание

Этот проект представляет собой веб приложение для анализа снимков компьютерной томографии (КТ).
Пользователь может загрузить исследование КТ в формате DICOM через drag&drop, чтобы получить размеченное изображение, 
если на нем присутсвуют подозрительные образования. Реализуемое ПО призвано увеличить число обнаружений рака молочной
железы для более раннего лечения, а также работать как система помощи принятия медицинских решений.

## Цель проекта

Разработать веб приложение, которое будет сегментировать рак молочной железы по исследованиям компьютерной томографии в формате DICOM.

## Правила именования

### Ветки
`<временная_ветка>/<область>-<описание>`
- Временные_ветки: features, bugfix, hotfix, release
- Области: analysis, backend, frontend, devops, docs, deps, core
- Пример: `features/analysis-train-model`

### Коммиты
`<тип>(<область>): <описание>`
- Типы: feat, fix, docs, style, refactor, test, chore
- Области: analysis, backend, frontend, devops, docs, deps, core
- Пример: `feat(analysis): добавить модель CNN`

### Pull Request
`<тип>(<область>): <описание>`
- Типы: feat, fix, docs, style, refactor, test, chore
- Области: analysis, backend, frontend, devops, docs, deps, core
- Пример: `feat(backend): обучение CNN модели`
